# rai413
RAI - Redis Automate Instance. Initialize, delete or override existing instance of Redis. Create new data space, config file, users and generate new connections. Write everything in YAML file. When you open specific version on release list, for example version v1.0, inside of that package you can find EULA agreement, official documentation in PDF format and packages of binary files, separated according to operating system for which binary files are built. All parts of 413 ecosystem you can find on https://www.413.world.
